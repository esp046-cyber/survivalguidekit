# survivalguidekit
Features:
Survival theme with military-green camouflage colors and tactical styling
Small font titles as requested (1.2em for main, 0.7em subtitle)
1% margins throughout for minimal spacing
6 categories: Core Essentials, Water & Food, Shelter & Fire, First Aid, Tools, Navigation
Priority system: Color-coded HIGH/MEDIUM/LOW priorities for each item
Practical tips in each section with survival advice
Mobile-friendly: Responsive grid layout, touch-optimized buttons
No search bar or 3D elements as specified
The app covers essential survival knowledge for beginners including the Rule of Threes, water purification, shelter building, fire starting, first aid, and navigation basics. Each item includes practical descriptions to help beginners understand what to pack and why.
